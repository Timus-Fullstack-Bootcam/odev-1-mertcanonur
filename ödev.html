5.     NaN (Not a Number), matematiksel bir işlem sonucunda geçerli bir sayısal değer üretilemediğinde ortaya çıkan bir özel değerdir. Bu durum, özellikle sayısal bir operasyonun geçersiz veya tanımsız olduğu durumlarda, örneğin sıfıra bölme veya sayısal olmayan bir değeri sayısal bir değere çevirme girişiminde meydana gelebilir. NaN, sayısal olmayan bir değeri temsil eder ve bu değerle gerçekleştirilen matematiksel operasyonlar genellikle NaN sonuçları üretir. Bu durumu kontrol etmek için isNaN() fonksiyonu kullanılabilir.

9.  == ile === arasındaki temel fark tip dönüşümüdür. == sadece değerleri karşılaştır. Type önemli değildir. === ise hem değerleri hem typlerı karşılaştırır. Type önemlidir

       Örnek vermek gerekirse

'5' == 5;  // true, türler dönüşümü yapılır, değerler eşit

'5' === 5;  // false, türler farklı olduğu için eşit değiller

    15. Object destructuring, bir nesnenin içindeki özellikleri çıkarmak veya ayırmak için kullanılan bir JavaScript özelliğidir. Bu özellik, özellikle nesnelerle çalışırken daha okunabilir ve kısa kodlar yazmanızı sağlar.

Örnek vermek gerekirse;

const user = {

name = “mertcan”,

surname = “Onur”,

age = “23”

};

Object destructuring ile yapılma şekli ise şöyledir;

const { name,surname,age } = user;

console.log(name);

console.log(surname);

console.log(age);
